# Keyboard Shortcuts

Trimmy keeps playback and trimming shortcuts close to common video-player keys.

| Key | Action |
| --- | --- |
| `K` | Play or pause |
| `J` | Seek backward 5 seconds |
| `L` | Seek forward 5 seconds |
| `Q` | Set trim start to playhead |
| `E` | Set trim end to playhead |
| `M` | Toggle mute |
| `?` | Show keyboard shortcuts |

## Fast trim workflow

1. Press `K` to play.
2. Press `Q` at the first frame you want.
3. Press `E` at the last frame you want.
4. Adjust the crop boxes while watching the output preview.
5. Click `Render Video` or `Send to Queue`.
